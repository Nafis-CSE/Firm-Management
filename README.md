# 🏢 Firm Management System (C++ Console Project)

A simple **console-based Firm Management System** built using **C++**, designed to manage basic business operations like income, expense, employee records, and stock management.

---

## 🔐 Login / Sign Up System

At the start of the program, users must:

- Login to their account  
- OR Sign up for a new account  

Only authenticated users can access the main system.

---

## 📊 Main Menu (After Login)

After successful login, the system shows a **menu-driven interface** with 4 main options:

### 💰 1. Income
- Add income records  
- View total income  
- Track business earnings  

### 💸 2. Expense
- Add expense records  
- View total expenses  
- Manage cost details  

### 👨‍💼 3. Employee
- Add new employee  
- View employee list  
- Manage employee data  

### 📦 4. Stocks
- Add stock items  
- Update stock quantity  
- Track available products
